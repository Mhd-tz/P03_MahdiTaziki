Overview

Listen to the World

The CJSF 90.1 FM application allows users and listeners to tune into On Air programs on CJSF 90.1 FM. The project started as a side project and was later expanded into a full-blown web application, bringing live radio streaming and on-demand podcast content to a wider audience.

  • React Native
  • UI/UX design
My Role

Front-End Development

As a front-end developer, I was responsible for developing the mobile application using React Native. I also worked on the UI/UX design for the application to increase user engagement and conversion rate. The project involved integrating real-time streaming APIs, managing audio state, and designing animated elements for a lively listening experience.
